# 1.字符串分割

## 描述

    给定一个**string**,以及若干种划分字符，要求将该string以划分字符为界分割成若干子字符串

## 模板实现

```
void split(string &s,vector<string> &res){
    string tmp;
    tmp.clear();
    for(int i=0;i<s.size();i++){
        if(s[i]==',') {//使用时在此处添加分割的特殊字符
            res.push_back(tmp);
            tmp.clear();
        }
        else tmp+=s[i];
    }
    res.push_back(tmp);
    tmp.clear();
}
```

## 简要说明

    该模板接收要划分的字符串，将划分后的结果存放到**string**数组中。使用时在注释处修改划分字符即可。
